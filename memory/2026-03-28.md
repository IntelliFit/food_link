## 精准模式下「上传图片失败」
- 代码侧：`execution_mode` 不参与 `/api/upload-analyze-image`，上传与模式无直接耦合；更可能是精准模式用户按提示拍得更清晰/更大 → base64 JSON 体积超网关限制，或错误响应无 `detail` 时前端只显示泛化「上传图片失败」
- 已做：`compressImagePathForUpload`（`wx.compressImage`）在分析页转 base64 前压缩；`uploadAnalyzeImage` 延长超时、可选带 Bearer、413/HTTP 状态友好提示
